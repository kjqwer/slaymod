# MySts2Mod

这是一个以**储君增强**为主题的 Slay the Spire 2 模组。

## 模组定位

- 强化储君（Regent）体系，补充更多围绕星辰、碎屑、锻造与手牌管理的玩法
- 提供更清晰的流派拼装方向，增强中后期构筑上限
- 聚焦储君专属构筑与战斗节奏优化

## 本次主要改动

- 新增卡牌 **Overload Dumping**（过载倾泻）
  - 2费稀有攻击
  - 对所有敌人造成伤害
  - 每有1点星辰，向手牌加入1张碎屑并追加额外群体伤害
  - 升级后提高基础伤害与额外伤害
- 皇家导师委任状（TheRoyalTutorsAssignmentRelic）调整
  - 稀有遗物
  - 归入储君遗物池（RegentRelicPool）
  - 回合开始获得1点能量与1点星辰
  - 若回合结束仍有剩余能量或星辰，则施加1层虚弱与1层易伤
