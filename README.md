# MySts2Mod

这是一个以**储君增强**为主题的 Slay the Spire 2 模组，同时包含少量原版卡牌的平衡调整。

## 模组定位

- 强化储君（Regent）体系，补充更多围绕星辰、碎屑、锻造与手牌管理的玩法
- 提供更清晰的流派拼装方向，增强中后期构筑上限
- 对部分卡牌做平衡调整，优化实战体验

## 本次主要改动

- 新增卡牌 **Overload Dumping**（过载倾泻）
  - 2费稀有攻击
  - 对所有敌人造成伤害
  - 每有1点星辰，向手牌加入1张碎屑并追加额外群体伤害
  - 升级后提高基础伤害与额外伤害
- 平衡调整原版 **Relax**
  - 费用调整为 2
  - 格挡调整为 20
  - 升级后格挡为 30
